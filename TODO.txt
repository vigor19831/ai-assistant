## Часть 2: Sacred Core → Stable Core

**Проблема:** Глобальный registry, frozen pipeline, dynamic router.

**Решение:**

| Компонент | Сейчас | Должно быть |
|-----------|--------|-------------|
| `registry.py` | Глобальный `_registry` dict | `AdapterRegistry` class, injectable |
| `pipeline.py` | Sequential only | Configurable: error handling, retry, parallel |
| `router.py` | `importlib` dynamic | Explicit registration registry |
| `ports/` | Flat | Versioned: `ports/v1/`, `ports/v2/` |

**Код registry:**
```python
class AdapterRegistry:
    def __init__(self) -> None:
        self._registry: dict[str, dict[str, type]] = {}

    def register(self, port: str, name: str, cls: type) -> None:
        self._registry.setdefault(port, {})[name] = cls

    def create(self, port: str, name: str, config: Any) -> Any:
        if port not in self._registry or name not in self._registry[port]:
            raise ValueError(f"No adapter: {port}/{name}")
        return self._registry[port][name](config)

    def fork(self) -> "AdapterRegistry":
        new = AdapterRegistry()
        new._registry = {k: dict(v) for k, v in self._registry.items()}
        return new

# Глобальный для @register декоратора
_default_registry = AdapterRegistry()

def register(port: str, name: str) -> Callable[[type], type]:
    def decorator(cls: type) -> type:
        _default_registry.register(port, name, cls)
        return cls
    return decorator
```

**Загрузка:**
```python
from pathlib import Path
import yaml
from pydantic import BaseModel

class ConfigLoader:
    def __init__(self, env: str = "production") -> None:
        self.env = env

    def load(self) -> AppConfig:
        base = Path("config/config.yaml")
        override = Path(f"config/config.{self.env}.yaml")

        data = yaml.safe_load(base.read_text())
        if override.exists():
            data |= yaml.safe_load(override.read_text())

        return AppConfig(**data)
```
